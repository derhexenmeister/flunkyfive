*** RISC V Software Development ***

The picorv32 project documents how to download and build the RISC V software development tools.

Alternatively prebuilt RISC V tools can be grabbed from the sifive site. These are provided for use with the Arduino tools.

See the list of options in:

    https://static.dev.sifive.com/bsp/arduino/package_sifive_index.json

I chose to try these:

    http://static.dev.sifive.com/bsp/arduino/riscv32-unknown-elf-gcc-e3e50c50-x86_64-pc-linux-gnu.tar.bz2

Also for Raspberry Pi see:

    http://icoboard.org/get-started-with-your-icoboard-and-a-raspi.html
